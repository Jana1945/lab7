# Logistic Regression Assignment

## Overview
This project uses Logistic Regression to predict whether a user will click on an advertisement.

## Dataset
The dataset (advertising.csv) includes:
- Age
- Daily Time Spent on Site
- Area Income
- Daily Internet Usage
- Clicked on Ad (target)

## Steps
1. Load the data
2. Explore the data using plots
3. Select important features
4. Split data into training and testing
5. Train Logistic Regression model
6. Evaluate using classification report

## Notes
Only numerical features were used. Data was split 70% training and 30% testing.
